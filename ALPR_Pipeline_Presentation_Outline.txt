Slide 6: Detailed Pipeline Steps
-------------------------------
- Step 1: Image Upload
  - User uploads a vehicle image via Streamlit interface.
- Step 2: Condition Classification
  - Classifies scene (e.g., lighting, blur) to adapt pipeline.
- Step 3: NAFNet Deblurring
  - Always deblurs image to improve detection accuracy.
- Step 4: YOLOv8 Detection
  - Detects license plates on the deblurred image.
- Step 5: DarkIR Fallback
  - If detection fails, applies DarkIR for further restoration and retries detection.
- Step 6: Plate ROI Extraction & Deblurring
  - Crops detected plate, applies NAFNet again for best OCR results.
- Step 7: OCR Preprocessing
  - Enhances plate image (contrast, resizing, etc.).
- Step 8: Text Extraction
  - Uses EasyOCR to extract license plate text.
